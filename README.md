# Moscorps

**Your backend, already written.**

The "create-react-app" moment for backends — industry-specific templates with real-world schemas, business logic, auth flows, and APIs already baked in.

## Quick Start

```bash
npx moscorps create my-app --db postgres --auth jwt --template saas
```

## Templates

- **SaaS** - Multi-tenancy, subscriptions, team management
- **E-commerce** - Products, cart, orders, inventory
- **Agency** - Clients, projects, invoices
- **Landing** - Leads, waitlists, analytics
- **Marketplace** - Buyers, sellers, listings
- **LMS** - Courses, students, progress tracking

## Installation

```bash
npm install -g moscorps
```

## Features

- ✅ TypeScript by default
- ✅ Multiple databases (PostgreSQL, MySQL, SQLite, MongoDB)
- ✅ JWT & OAuth authentication
- ✅ Role-based access control
- ✅ RESTful API generation
- ✅ Admin panel ready
- ✅ File storage support
- ✅ Plugin system
