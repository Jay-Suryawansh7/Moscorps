# Contributing to Moscorps

## Getting Started

1. Fork the repository
2. Clone your fork: `git clone https://github.com/YOUR_USERNAME/moscorps.git`
3. Install dependencies: `npm install`
4. Create a branch: `git checkout -b feature/your-feature`

## Development

```bash
# Install dependencies
npm install

# Run tests
npm test

# Build
npm run build
```

## Adding a Template

1. Create template in `src/templates/your-template/`
2. Follow existing template structure
3. Add tests
4. Update documentation

## Commit Messages

We use conventional commits:
- `feat:` New feature
- `fix:` Bug fix
- `docs:` Documentation
- `chore:` Maintenance
- `test:` Tests

## Pull Requests

1. Describe your changes
2. Add tests if applicable
3. Update documentation
4. Ensure CI passes

## Code of Conduct

- Be respectful
- Be constructive
- Be inclusive

## Questions?

Open an issue for questions or discussions.
