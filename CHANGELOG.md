# Changelog

All notable changes to Moscorps will be documented in this file.

## [0.1.0] - 2024-04-14

### Added
- Initial release
- CLI with `create` command
- TypeScript support
- 6 industry templates:
  - SaaS (multi-tenancy, subscriptions)
  - E-commerce (products, cart, orders)
  - Agency (clients, projects, invoices)
  - Landing (leads, waitlists)
  - Marketplace (listings, transactions)
  - LMS (courses, enrollments)
- JWT authentication
- Role-based access control
- RESTful API generation
- Database support (PostgreSQL, MySQL, SQLite, MongoDB)
